# Icons

Icons are telegraphic visual shortcuts that can symbolize people, ideas, objects, places, activities, and themes. They have the ability to transcend linguistic and cultural differences to help us communicate universally.

## Usage

See the <a onClick={linkTo("hello-getting-started--page")}>Getting Started</a> documentation to download or install icons.

When you download or install the icons, you can do a simple copy and paste, setup an automated task to move the files out of node_modules or you can view the icon in the browser by clicking on the download link and copy the code by viewing the source of the page.

We recommend inlining SVG when possible since it's a best practice. Overall, this is the simplest way for using SVG in your HTML documents. Simply copy the HTML from the icon file you need and paste it into your app or website.

**CSS overrides**  
You can override how an SVG icon looks using CSS.

```
<svg
  width="24"
  height="24"
  viewBox="0 0 24 24"
  class="icon"
  xmlns="http://www.w3.org/2000/svg"
>
<title>Close Button</title>
  <path
    fill="currentColor"
    d="M22.753 5.394 24 6.645 12 18.606 0 6.646l1.247-1.252L12 16.11z"
    fill-rule="evenodd"
  />
</svg>
```

```
.icon {
  height: var(--made-size-4-x);
  width: var(--made-size-4-x);
  fill: var(--made-color-text-default-on-light);
}
```

**Accessibility**  
For screen reader accessibility, use `<title>` element.
The `<title>` element describes the SVG and what it's used for. Make this as detailed as possible for screen-readers and overall accessibility.
